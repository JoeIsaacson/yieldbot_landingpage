Cookie-less Targeting Riding the world of bad data

3rd party cookie data has been in use for many years. First pioneered by companies like Tacoda the use of cookies to write and store information about people’s web browsing behavior for ad targeting.

Selling Your Browsing History

More recently 3rd party businesses have been created to track people and sell the cookies they have collected to another party all together. Many times there is not enough cookie data on certain web behaviors.

In that case the attributes of one data pool are used to find similar data in other cookie pools to create a set of “Look-alike” cookies that are used for ad targeting.

Retargeting

Another practice of 3rd party data is in re-targeting. We’ve all seen ads for trips we have already taken. Shoes we’ve already bought. That same ad that creeps around while you surf the web.

Using cookies to track you has never increased the value of digital media or delivered significant performance improvements for advertisers.
